Big-O

Hierarchy

Constant O(1)
  - indexing into array

Logarithmic O(log n)
  - binary search
  - divide and conquer algorithms

Linear O(n)
  - array, string iteration

Linearithmic/Log-Linear O(n log n)
  - merge sort

Quadratic O(n^2)
  - nested loops
  - quick sort (worst case, O(n log n) avg case)
  - bubble sort

Exponential O(k^n)
  - opposite of log, given input size, each step doubles size
  - ex: subsets

Factorial O(n!)
  - permutations
  - n! = n*(n-1)!
